# Security Policy

## Supported Versions

We support the latest version of UptimeKit Worker.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of UptimeKit seriously. If you believe you have found a security vulnerability in the UptimeKit Worker, please report it to us by following these steps:

1. **Do not open a public issue.**
2. Send an email to **<security@uptimekit.com>** with details of the vulnerability.
3. Include as much information as possible, including:
    * Steps to reproduce the vulnerability.
    * A description of the potential impact.
    * Any possible mitigations.

We will acknowledge receipt of your report and provide a timeline for addressing the issue. We ask that you follow responsible disclosure practices and give us time to fix the vulnerability before making it public.

Thank you for helping keep UptimeKit secure!
